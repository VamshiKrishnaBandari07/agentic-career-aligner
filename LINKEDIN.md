## Option 1 (story style — recommended)

I was tired of paying for CV review tools while job hunting.

So I built my own — **Career Aligner**.

Upload your resume PDF. Paste any job description. Get instant feedback:
- Match score
- Missing skills
- What to change on your CV
- Actionable resume suggestions

**100% free. Runs on your laptop. No subscription. No API key. Your CV never leaves your machine.**

I used it for my own applications. Now I'm sharing it with everyone.

Clone → run → review your CV in 2 minutes:

```
git clone https://github.com/VamshiKrishnaBandari07/agentic-career-aligner.git
cd agentic-career-aligner
run.bat        # Windows (double-click works too)
# ./run.sh     # Mac/Linux
```

Open http://127.0.0.1:8000 — that's it.

Built with Python + FastAPI. Open source (MIT).

If you're applying for jobs right now, this is for you.

GitHub: https://github.com/VamshiKrishnaBandari07/agentic-career-aligner
